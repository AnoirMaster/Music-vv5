📁 Setup Instructions
──────────────────────────────

1️⃣  INSTALL REQUIRED MODULES
──────────────────────────────
➡️  Option A: Run the following command in terminal:
    > npm install

➡️  Option B: Simply double-click:
    > install_modules.bat

2️⃣  CONFIGURE THE BOT
──────────────────────────────
🛠️  Go to the config file:
    > Music Bot V5.0.1\settings\config.js

✏️  Edit the following fields:
    - TOKENS (your Discord bot tokens)
    - OWNER_ID (your Discord user ID for owner commands)
    - EMBED_COLOR (color used in embed messages)
    - DEFAULT_VOLUME (e.g., 50 - the default volume when a new queue starts)
    - SPOTIFY_ID (your Spotify client ID for API authentication)
    - SPOTIFY_SECRET (your Spotify client secret for API authentication)
    - Any other optional settings

3️⃣  INSTALL SLASH COMMANDS
──────────────────────────────
➡️  Double-click:
    > install_slash_commands.bat

✅ This registers the commands with your Discord server.

4️⃣  START THE BOT
──────────────────────────────
➡️  Option A: Run from terminal:
    > node index.js

➡️  Option B: Double-click:
    > start.bat

🎉 That's it! Your bot should now be online and playing music!


📌 Notes
──────────────────────────────
- Make sure Node.js is installed (https://nodejs.org/)
- Configure your bot application properly at:
  https://discord.com/developers/applications
- Slash commands may take a few minutes to appear
